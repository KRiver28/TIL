# pull, clone

- 가정

  - 이미 레포지토리에 커밋이 올라가 있다.

  - pull, push하는 권한이 있다.

- clone

  - 빈 폴더( 상위 폴더에 깃이 없어야 함)에 clone을 합니다
    - `git clone {url}` -> 새로운 폴더를 만들면서
    - `git clone {url} .` -> 현재 폴더에 그대로

- do something and push commit
  - 원격 저장소가 원본 로컬 저장소보다 상위 버전이 됨.
- pull
  - 원본 로컬 저장소에서 원격 저장소의 코밋을 pull 함
    - `git pull`
