# 텍스트 강조

- `**안녕**` : 볼드체 **안녕**  `ctrl + b`
- `*안녕*` : 이태리체 *안녕* `ctrl + i`
- `~~안녕~~` :  취소선 ~~안녕~~ 
- `***안녕***` :볼드+이태리 ***안녕*** `ctrl+b+i`
